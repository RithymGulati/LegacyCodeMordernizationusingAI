# LegacyAI 

Link to the website https://legacyai-resume-project.vercel.app/

AI-powered legacy code analysis platform for understanding, documenting, and modernizing enterprise applications.

LegacyAI helps developers and modernization teams analyze legacy source code by generating inline explanations, business logic annotations, modernization recommendations, maintainability insights, and risk analysis using Large Language Models.

---

## Features

- AI-generated inline code annotations
- Business logic understanding
- Risk detection and maintainability scoring
- Modernization recommendations
- Side-by-side raw vs analyzed code view
- Multi-language legacy code support
- Professional dark-themed enterprise UI
- Structured AI insights dashboard
- Sample snippets for quick testing
- OpenRouter LLM integration
